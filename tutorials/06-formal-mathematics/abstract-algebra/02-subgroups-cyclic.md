# Subgroups and Cyclic Groups

## Intuition

A subgroup is a group hiding inside a bigger group — like how the even
integers form a group under addition inside all integers. A **cyclic group**
is one generated by repeatedly applying a single element, like how pressing
one rotation key on a Rubik's face cycles through 4 positions and back.

## From First Principles

### Subgroup Definition

A subset $H \subseteq G$ is a **subgroup** of $(G, \star)$, written $H \leq G$, if $(H, \star)$ is itself a group.

### One-Step Subgroup Test

$H \leq G$ if and only if:
1. $H \neq \emptyset$ (usually shown by checking $e \in H$).
2. $\forall a, b \in H,\; a \star b^{-1} \in H$.

Why this works: condition 2 gives closure (take $b^{-1}$ to get inverses, then compose), identity ($a \star a^{-1} = e$), and associativity is inherited from $G$.

### Two-Step Subgroup Test

$H \leq G$ if and only if:
1. $\forall a, b \in H,\; a \star b \in H$ (closure).
2. $\forall a \in H,\; a^{-1} \in H$ (inverses).

### Example: Even Integers

$2\mathbb{Z} = \{\ldots, -4, -2, 0, 2, 4, \ldots\} \leq (\mathbb{Z}, +)$.

Check: $0 \in 2\mathbb{Z}$. If $a, b \in 2\mathbb{Z}$, then $a - b$ is even, so $a - b \in 2\mathbb{Z}$. Done.

### Cyclic Groups

The **cyclic group generated by $g$** is:

$$\langle g \rangle = \{g^n : n \in \mathbb{Z}\} = \{\ldots, g^{-2}, g^{-1}, e, g, g^2, \ldots\}$$

If $G = \langle g \rangle$, we say $G$ is **cyclic** and $g$ is a **generator**.

### Order of an Element

The **order** of $g \in G$, written $|g|$ or $\text{ord}(g)$, is the smallest positive integer $n$ such that $g^n = e$. If no such $n$ exists, $|g| = \infty$.

**Example** in $(\mathbb{Z}_6, +)$:
- $|0| = 1$ (already identity)
- $|1| = 6$ ($1+1+\cdots = 0$ after 6 additions)
- $|2| = 3$ ($2, 4, 0$)
- $|3| = 2$ ($3, 0$)

### Pen & Paper: Subgroups of $\mathbb{Z}_{12}$

The cyclic group $\mathbb{Z}_{12}$ has subgroups for each divisor of 12:

| Divisor $d$ | Subgroup $\langle 12/d \rangle$ | Elements |
|------------|-------------------------------|----------|
| 1 | $\langle 12 \rangle = \{0\}$ | $\{0\}$ |
| 2 | $\langle 6 \rangle$ | $\{0, 6\}$ |
| 3 | $\langle 4 \rangle$ | $\{0, 4, 8\}$ |
| 4 | $\langle 3 \rangle$ | $\{0, 3, 6, 9\}$ |
| 6 | $\langle 2 \rangle$ | $\{0, 2, 4, 6, 8, 10\}$ |
| 12 | $\langle 1 \rangle$ | $\mathbb{Z}_{12}$ |

**Theorem**: Every subgroup of a cyclic group is cyclic.

### Visualisation

```python
import matplotlib.pyplot as plt
import numpy as np

# Visualise Z_12 as a clock and its subgroups
fig, axes = plt.subplots(2, 3, figsize=(12, 8))

n = 12
theta = np.linspace(0, 2 * np.pi, n, endpoint=False)
xs = np.sin(theta)
ys = np.cos(theta)

subgroups = [
    ([0], '<0> = {0}'),
    ([0, 6], '<6> = {0,6}'),
    ([0, 4, 8], '<4> = {0,4,8}'),
    ([0, 3, 6, 9], '<3> = {0,3,6,9}'),
    ([0, 2, 4, 6, 8, 10], '<2>'),
    (list(range(12)), '<1> = Z_12'),
]

for ax, (sg, title) in zip(axes.flat, subgroups):
    ax.plot(xs, ys, 'o', color='lightgray', markersize=10)
    for i in range(n):
        ax.text(xs[i] * 1.15, ys[i] * 1.15, str(i),
                ha='center', va='center', fontsize=8)
    sg_xs = [xs[i] for i in sg]
    sg_ys = [ys[i] for i in sg]
    ax.plot(sg_xs, sg_ys, 'ro', markersize=12)
    if len(sg) > 1:
        for i in range(len(sg)):
            j = (i + 1) % len(sg)
            ax.plot([sg_xs[i], sg_xs[j]], [sg_ys[i], sg_ys[j]],
                    'r-', linewidth=1.5)
    ax.set_xlim(-1.5, 1.5)
    ax.set_ylim(-1.5, 1.5)
    ax.set_aspect('equal')
    ax.set_title(title, fontsize=10)
    ax.axis('off')

plt.suptitle('Subgroups of Z₁₂', fontsize=14)
plt.tight_layout()
plt.savefig('z12_subgroups.png', dpi=100)
plt.show()
```

## Python Verification

```python
# ── Subgroups and Cyclic Groups ──────────────────────────

# Step 1: Find all subgroups of Z_n
def cyclic_subgroup(g, n):
    """Generate <g> in Z_n."""
    H = set()
    current = 0
    while True:
        H.add(current)
        current = (current + g) % n
        if current in H:
            break
    return H

n = 12
print(f"=== Subgroups of Z_{n} ===")
subgroups = set()
for g in range(n):
    H = frozenset(cyclic_subgroup(g, n))
    subgroups.add(H)

for H in sorted(subgroups, key=len):
    print(f"  |H|={len(H):2d}: {sorted(H)}")

print(f"\nNumber of subgroups: {len(subgroups)}")
print(f"Divisors of {n}: {[d for d in range(1, n+1) if n % d == 0]}")

# Step 2: Compute order of every element in Z_n
print(f"\n=== Element Orders in Z_{n} ===")
for g in range(n):
    order = 1
    current = g
    while current % n != 0 or order == 0:
        if (g * order) % n == 0:
            break
        order += 1
    print(f"  |{g}| = {order}")

# Step 3: Verify subgroup test for even integers in Z_12
print(f"\n=== Subgroup test: even elements of Z_{n} ===")
H = {x for x in range(n) if x % 2 == 0}
print(f"H = {sorted(H)}")

# One-step test: a - b mod n in H for all a, b in H
is_subgroup = True
for a in H:
    for b in H:
        if (a - b) % n not in H:
            is_subgroup = False
            print(f"  FAIL: {a} - {b} = {(a-b)%n} not in H")
print(f"H is subgroup: {is_subgroup}")

# Step 4: Find generators of Z_n (elements with order n)
print(f"\n=== Generators of Z_{n} ===")
from math import gcd
generators = [g for g in range(n) if gcd(g, n) == 1]
print(f"Generators (gcd(g,{n})=1): {generators}")
print(f"Number of generators = phi({n}) = {len(generators)}")

# Step 5: Verify theorem: |<g>| = n / gcd(g, n)
print(f"\n=== |<g>| = n / gcd(g, n) ===")
for g in range(n):
    sg = cyclic_subgroup(g, n)
    expected = n // gcd(g, n)
    print(f"  g={g:2d}: |<g>|={len(sg):2d}, n/gcd(g,n)={expected:2d}, "
          f"match={len(sg)==expected}")
```

## Connection to CS / Games / AI / Business / Industry

- **Cyclic redundancy check (CRC)**: uses cyclic groups over polynomial rings to detect transmission errors.
- **Clock arithmetic**: scheduling, circular buffers, and hash functions use $\mathbb{Z}_n$.
- **Rubik's Cube groups**: each face generates a cyclic subgroup; the full group is generated by 6 face rotations.
- **Cryptographic key sizes**: the security of discrete-log based systems depends on the order of the generator element.
- **CRC-32 in Ethernet/ZIP/PNG** — uses cyclic subgroups of $\mathbb{Z}_2[x]/(g(x))$ where $g(x)$ is the IEEE 802.3 polynomial; Intel CPUs since Nehalem (2008) have a hardware `CRC32` instruction for this.
- **Bitcoin/Ethereum private key generation** — picks a uniformly random integer in the cyclic group $\langle G \rangle$ of order $\approx 2^{256}$ on secp256k1; the entire $43 trillion crypto market depends on this group's prime order.
- **5G NR Polar codes** (3GPP standard) — generated via cyclic shifts of a kernel matrix, exploiting cyclic subgroups for hardware-friendly decoding in Qualcomm Snapdragon modems.
- **Phasor analysis in power grids** — three-phase AC voltages form a cyclic group $\mathbb{Z}_3$ of $120°$ rotations; ABB and Siemens grid-stabilization SCADA systems exploit this symmetry to detect faults in milliseconds.
