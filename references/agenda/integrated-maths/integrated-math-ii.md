# Integrated Math II

> Master the algebra of quadratics and get acquainted with more advanced functions including polynomials, logarithms, trigonometry, and more. Learn core concepts in probability.

**Prerequisites:** Integrated Math I

## Learning Outcomes

Upon completing Integrated Math II, students will be able to:

- Solve quadratic equations using multiple methods and model with quadratic functions
- Work with function composition, inverses, and graph transformations
- Understand exponential function transformations and introduce logarithms
- Construct geometric proofs involving congruence, similarity, and circle theorems
- Calculate surface areas and volumes of three-dimensional solids
- Apply trigonometric ratios to right triangles and extend them using the unit circle
- Work with radians and special trigonometric ratios
- Apply probability concepts and analyze bivariate data using correlation
